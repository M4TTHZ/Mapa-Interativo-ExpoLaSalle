Sobre o projeto

Construído por estudantes, para a comunidade

🎯

Objetivo

Uma aplicação web moderna e responsiva que permite aos visitantes do ExpoLaSalle explorar todos os pontos de interesse do evento através de um mapa intuitivo.

⚙️

Tecnologias

  • HTML5 semântico
  • CSS3 moderno (Grid + Flex)
  • JavaScript (ES6+)
  • Leaflet.js + OpenStreetMap
💡

Por quê

  • Leaflet leve e customizável
  • Mobile-first responsivo
  • Performance e acessibilidade
  • Sem dependências de build
