Sobre o Projeto

Objetivo

Criar uma aplicação web moderna e responsiva que permite aos visitantes do ExpoLaSalle explorar interativamente todos os pontos de interesse do evento através de um mapa intuitivo e visualmente atraente.

Tecnologias Utilizadas

  • HTML5: Estrutura semântica e acessível
  • CSS3: Design responsivo com Flexbox e Grid
  • JavaScript: Interatividade e lógica da aplicação
  • Leaflet.js: Biblioteca de mapeamento interativo
  • OpenStreetMap: Dados de mapa de código aberto

Motivos das Escolhas Técnicas

  • Leaflet.js: Leve, flexível e fácil de customizar
  • Vanilla JavaScript: Sem dependências pesadas, melhor performance
  • CSS Grid + Flexbox: Layout moderno e responsivo
  • Mobile First: Otimizado para todos os dispositivos
